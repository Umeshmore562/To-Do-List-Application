To-Do List Application

The To-Do List Application is a lightweight productivity tool designed to help users organize their tasks efficiently. 
It features a clean, modern interface created with CustomTkinter, offering a futuristic dark theme that enhances focus and user experience. 
Users can add tasks quickly, mark them as completed, or delete them with a single click. 
All tasks are automatically stored in a local JSON file, ensuring that entries remain intact even after restarting the program.
